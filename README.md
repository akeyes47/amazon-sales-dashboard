Amazon Sales Dashboard

Overview  
This project analyzes e-commerce sales data using Excel to identify trends in revenue, profitability, and customer behavior. An interactive dashboard was created to visualize key business metrics and support data-driven insights.

Features  
- Interactive dashboard with filters for:
  - Product Category  
  - Customer Region  
  - Payment Method  
- Monthly analysis of:
  - Revenue  
  - Profit  
- Key performance indicators:
  - Total Revenue  
  - Total Profit  
  - Average Rating  

Key Insights  
- Revenue remains relatively stable across months  
- Despite strong revenue performance, the business operates at a loss  
- Discounts appear to significantly impact profitability  
- Some product categories generate revenue but not sustainable profit  

Tools Used  
- Microsoft Excel  
- Pivot Tables  
- Data Visualization  
- Data Cleaning and Transformation  

Files  
- dashboard/amazon-sales-dashboard.xlsx → Interactive dashboard  
- data/amazon_sales.csv → Raw dataset
